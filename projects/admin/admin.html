Stripe Integration

To connect Stripe checkout:

  1. Create a Stripe account at stripe.com
  2. Add your STRIPE_SECRET_KEY to your server environment
  3. Create a /api/checkout endpoint that creates a Stripe session
  4. Add your webhook endpoint and verify the stripe-signature header
  5. Set STRIPE_WEBHOOK_SECRET in your environment
Stripe Quickstart Guide

Google OAuth Setup

To enable Google login:

  1. Go to console.cloud.google.com
  2. Create a project → APIs & Services → Credentials
  3. Create an OAuth 2.0 Client ID (Web application)
  4. Add your domain to Authorised JavaScript Origins
  5. Replace YOUR_GOOGLE_CLIENT_ID in admin.js
